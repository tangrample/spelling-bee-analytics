🐝 Spelling Bee Bookmarklet

One-time setup on your iPhone — takes about 2 minutes.

1 Bookmark any page in Safari
Open Safari and tap the Share button (box with arrow). Tap Add Bookmark. Name it 🐝 Save Bee Data and save it anywhere.
2 Copy the bookmarklet code
Tap the button below to copy the code. You'll paste it in the next step.
javascript:(function(){var d={};for(var i=0;i<localStorage.length;i++){var k=localStorage.key(i);var v=localStorage.getItem(k);if(k.indexOf('spelling-bee')!==-1||k.indexOf('sb-')!==-1||k.indexOf('games-state')!==-1){try{d[k]=JSON.parse(v);}catch(e){d[k]=v;}}}try{if(typeof window.gameData!=='undefined')d['_window_gameData']=window.gameData;}catch(e){}var s=JSON.stringify(d,null,2);var b=new Blob([s],{type:'application/json'});var u=URL.createObjectURL(b);var a=document.createElement('a');a.href=u;a.download='spelling_bee_raw_'+new Date().toISOString().split('T')[0]+'.json';document.body.appendChild(a);a.click();document.body.removeChild(a);URL.revokeObjectURL(u);})()
3 Replace the bookmark URL with the code
In Safari, tap the bookmarks icon (open book). Find 🐝 Save Bee Data, swipe left and tap Edit. Delete everything in the URL field and paste the code you just copied. Tap Done.
💡 The URL field will look broken — that's normal for a bookmarklet.
4 Test it on the Spelling Bee page
Go to nytimes.com/puzzles/spelling-bee in Safari. Tap your 🐝 Save Bee Data bookmark. A download prompt will appear — tap Download then open the Downloads icon and tap the file. Choose Share → Save to Files → iCloud Drive → Spelling Bee.
💡 iOS remembers the last folder you saved to, so after the first time it's just 2 taps.

✅ Your daily routine (after setup)

  1. Finish playing Spelling Bee on your iPhone
  2. Tap 🐝 Save Bee Data bookmark
  3. Tap Download → Save to Files → Spelling Bee folder
  4. Done — your Mac processes it automatically 🎉
