🚀 2-Minute Setup: Keyboard Shortcut

Set it up once, then just press a key combo every day!
1 Open Shortcuts App

Press ⌘ Cmd + Space, type "Shortcuts", press Enter

2 Create New Shortcut

Click + New Shortcut in the top-right corner

3 Add "Run Shell Script" Action

Search for "Run Shell Script" in the actions panel and add it

cd ~/Desktop/spelling-bee-analytics/files ./auto_update_with_notification.sh

Paste this code into the script box

4 Name Your Shortcut

Click on "Untitled Shortcut" at the top and rename to: Update Spelling Bee

5 Add Keyboard Shortcut

Click the (i) icon → Add Keyboard Shortcut
Press: ⌃ Control + ⌥ Option + B

✅Setup Complete!

Your shortcut is ready. Now you'll get a notification when it's done!

📅 Your New Daily Routine (10 seconds total)

  1. Play Spelling Bee in Safari 🐝
  2. Click your bookmarklet 📥
  3. Press ⌃⌥B ⌨️
  4. See notification "✅ Updated" 🔔

Done! History is building automatically.

🎯 Even Faster: Menu Bar Option

Want a visual button instead of keyboard shortcut?

  1. In Shortcuts app, right-click your shortcut
  2. Select "Add to Menu Bar"
  3. Now you have a 🐝 icon in your menu bar - click anytime!
